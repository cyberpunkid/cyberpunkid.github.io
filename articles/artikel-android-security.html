10 Pengaturan Keamanan yang Harus Aktif di Android Anda Sekarang

Ponsel Android Anda menyimpan password, data perbankan, foto pribadi, dan percakapan rahasia — tapi mungkin belum seaman yang Anda kira. Aktifkan 10 pengaturan ini sekarang, tanpa perlu keahlian teknis.

// MULAI TRANSMISI

Android Aman? Tidak Otomatis

Android adalah sistem operasi mobile yang paling banyak digunakan di dunia — dan karenanya, juga yang paling banyak diserang. Banyak pengguna tidak menyadari bahwa default settings Android tidak selalu dikonfigurasi untuk keamanan maksimal. Beberapa pengaturan kritis harus diaktifkan secara manual.

Panduan ini menggunakan referensi Android stock (Pixel/AOSP). Letak menu mungkin sedikit berbeda di Samsung (One UI), Xiaomi (MIUI), atau brand lain, tapi nama pengaturannya sama.

1. Aktifkan Google Play Protect

Play Protect adalah antivirus bawaan Android dari Google yang memindai semua aplikasi secara otomatis.

Cara aktifkan: Buka Play Store → tap foto profil → Play Protect → Aktifkan "Scan apps with Play Protect" dan "Improve harmful app detection".

2. Nonaktifkan "Install dari Sumber Tidak Dikenal"

Pengaturan ini, jika aktif, mengizinkan instalasi APK dari luar Play Store — yang merupakan pintu masuk utama malware di Android.

Cara cek: Settings → Apps → tap titik tiga di kanan atas → Special app access → Install unknown apps → Pastikan semua aplikasi dalam kondisi "Not allowed".

// Perhatian

Jika Anda pernah install APK dari WhatsApp atau situs web, browser atau WhatsApp Anda mungkin masih punya izin untuk install app dari sumber tidak dikenal. Periksa dan cabut izin tersebut sekarang.

3. Audit Izin Aplikasi (App Permissions)

Banyak aplikasi meminta izin yang jauh melebihi kebutuhan fungsionalnya. Aplikasi senter yang meminta akses kontak, atau game yang meminta akses SMS — ini tanda-tanda aplikasi bermasalah.

Cara audit: Settings → Privacy → Permission Manager → Review setiap kategori (Location, Camera, Microphone, Contacts, SMS) dan cabut izin yang tidak relevan.

Perhatian khusus pada:

  • Microphone & Camera — hanya aplikasi yang memang butuh (kamera, video call) yang boleh punya akses
  • Location — set ke "Only while using the app", bukan "Always"
  • SMS — hanya aplikasi pesan default yang perlu akses SMS. Aplikasi lain yang meminta ini sangat mencurigakan
  • Accessibility — izin ini sangat powerful dan sering disalahgunakan malware. Hanya beri ke app yang benar-benar membutuhkan

4. Gunakan Screen Lock yang Kuat

PIN 4 digit dapat dibobol dalam hitungan menit. Gunakan PIN minimal 6 digit, atau lebih baik lagi, passphrase (kata sandi berupa kalimat pendek yang mudah diingat tapi sulit ditebak).

Cara set: Settings → Security → Screen Lock → pilih PIN (6 digit) atau Password. Aktifkan juga biometric (fingerprint/face) sebagai cara buka cepat, tapi pastikan PIN/password juga kuat sebagai fallback.

Aktifkan auto-lock setelah 30 detik atau 1 menit tidak digunakan.

5. Enkripsi Perangkat

Enkripsi memastikan data di ponsel Anda tidak dapat diakses tanpa screen lock, bahkan jika seseorang mengambil chip memori ponsel Anda secara fisik. Pada Android modern (Android 6+), enkripsi biasanya sudah aktif secara default.

Cara cek: Settings → Security → Encryption & credentials → pastikan "Phone is encrypted" atau "Encrypted".

6. Aktifkan Find My Device

Fitur ini memungkinkan Anda melacak, mengunci, atau menghapus data ponsel dari jarak jauh jika hilang atau dicuri.

Cara aktifkan: Settings → Security → Find My Device → aktifkan. Pastikan Location juga aktif. Test di android.com/find untuk memastikan berfungsi.

7. Update OS dan Aplikasi Secara Rutin

Pembaruan keamanan Android menutup celah (vulnerability) yang ditemukan. Penyerang aktif mengeksploitasi celah pada versi lama. Google merilis security patch bulanan — pastikan ponsel Anda selalu update.

Cara cek: Settings → System → System update. Juga aktifkan auto-update di Play Store: Play Store → Profile → Settings → Network preferences → Auto-update apps.

8. Aktifkan Autentikasi Dua Faktor di Akun Google

Akun Google adalah inti dari ekosistem Android Anda. Jika akun Google diretas, penyerang bisa mengakses backup data, email, bahkan reset akun lain. Aktifkan MFA di myaccount.google.com seperti yang dijelaskan di artikel terpisah kami.

9. Nonaktifkan Developer Options (Jika Tidak Dibutuhkan)

Developer Options berisi pengaturan tingkat lanjut yang bisa disalahgunakan jika ponsel Anda diakses orang lain. Jika Anda bukan developer, nonaktifkan fitur ini.

Cara nonaktifkan: Settings → System → Developer Options → nonaktifkan toggle di bagian atas. Jika menu Developer Options tidak ada, berarti sudah nonaktif.

Pastikan juga USB Debugging dalam kondisi off — fitur ini memungkinkan akses penuh ke perangkat via kabel USB dan sangat berbahaya jika aktif di luar kebutuhan development.

10. Gunakan DNS Privat

DNS Privat mengenkripsi query DNS Anda, mencegah ISP atau penyerang di jaringan yang sama memantau situs apa yang Anda kunjungi. Android 9+ mendukung DNS over TLS secara native.

Cara aktifkan: Settings → Network & Internet → Advanced → Private DNS → pilih "Private DNS provider hostname" → masukkan salah satu:

dns.google (Google DNS) one.one.one.one (Cloudflare DNS - direkomendasikan) dns.quad9.net (Quad9 - fokus keamanan, memblokir domain berbahaya)
// Rekomendasi

Quad9 (dns.quad9.net) adalah pilihan terbaik untuk keamanan ekstra karena secara otomatis memblokir domain yang diketahui berbahaya — termasuk domain phishing dan malware command-and-control.

Checklist Cepat

  1. ✅ Play Protect aktif
  2. ✅ Install unknown apps: semua "Not allowed"
  3. ✅ Izin app: sudah diaudit dan dibatasi
  4. ✅ Screen lock: PIN 6 digit atau passphrase
  5. ✅ Enkripsi: aktif
  6. ✅ Find My Device: aktif
  7. ✅ OS dan app: update terbaru
  8. ✅ Google Account MFA: aktif
  9. ✅ USB Debugging: nonaktif
  10. ✅ Private DNS: one.one.one.one atau dns.quad9.net

Android Is Not Automatically Secure

Android is the most widely used mobile OS in the world — and therefore the most targeted. Many users don't realize that Android's default settings aren't always configured for maximum security. Several critical settings must be manually enabled.

This guide references stock Android (Pixel/AOSP). Menu locations may differ slightly on Samsung (One UI), Xiaomi (MIUI), or other brands, but setting names are the same.

The 10 Settings to Activate Now

1. Enable Google Play Protect

Play Protect → Open Play Store → profile photo → Play Protect → Enable "Scan apps with Play Protect" and "Improve harmful app detection."

2. Disable "Install from Unknown Sources"

Settings → Apps → three dots → Special app access → Install unknown apps → Ensure all apps show "Not allowed."

// Warning

If you've ever installed APKs from WhatsApp or websites, your browser or WhatsApp may still have permission to install apps from unknown sources. Check and revoke that permission now.

3. Audit App Permissions

Settings → Privacy → Permission Manager → Review each category (Location, Camera, Microphone, Contacts, SMS) and revoke permissions that aren't relevant. Set Location to "Only while using the app." Only your default messaging app should have SMS access.

4. Use a Strong Screen Lock

Use a minimum 6-digit PIN or passphrase. Enable auto-lock after 30 seconds to 1 minute of inactivity.

5. Verify Device Encryption

Settings → Security → Encryption & credentials → ensure "Phone is encrypted" is shown. Most Android 6+ devices are encrypted by default.

6. Enable Find My Device

Settings → Security → Find My Device → enable. Test at android.com/find.

7. Keep OS and Apps Updated

Settings → System → System update. Enable auto-update in Play Store settings.

8. Enable MFA on Your Google Account

Visit myaccount.google.com and enable 2-Step Verification. Your Google account is the core of your Android ecosystem.

9. Disable Developer Options and USB Debugging

Settings → System → Developer Options → toggle off at the top. Ensure USB Debugging is off — it enables full device access via USB cable.

10. Enable Private DNS

Settings → Network & Internet → Advanced → Private DNS → hostname:

one.one.one.one (Cloudflare - recommended) dns.quad9.net (Quad9 - blocks malicious domains)

Quick Checklist

  1. ✅ Play Protect: enabled
  2. ✅ Unknown sources: all "Not allowed"
  3. ✅ App permissions: audited and limited
  4. ✅ Screen lock: 6-digit PIN or passphrase
  5. ✅ Encryption: active
  6. ✅ Find My Device: enabled
  7. ✅ OS and apps: latest update
  8. ✅ Google Account MFA: enabled
  9. ✅ USB Debugging: disabled
  10. ✅ Private DNS: configured
